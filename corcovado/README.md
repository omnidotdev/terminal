# Corcovado

Async I/O primitives for [Omni Terminal](https://github.com/omnidotdev/terminal). Uses Rust edition 2021 and the standard library for net/IO.

> Originally forked from [mio 0.6.x](https://github.com/tokio-rs/mio) with additions from [mio-signal-hook](https://github.com/vorner/mio-signal-hook) and [mio-extras](https://github.com/dimbleby/mio-extras), adapted via [Rio Terminal](https://github.com/raphamorim/rio) by [Raphael Amorim](https://github.com/raphamorim), licensed under MIT.
